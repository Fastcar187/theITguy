## üéØ How to Use

### Quick Start
1. Download or clone the project files
2. Open `index.html` in any modern web browser
3. The page will load with all animations and interactions active

### Customization

#### Colors
Edit the CSS custom properties in `styles.css`:
```css
:root {
    --base-navy: #0F172A;
    --primary-blue: #0369A1;
    --accent-violet: #818CF8;
    /* ... other colors */
}
```

#### Content
- Edit text content directly in `index.html`
- Replace placeholder product images in the Shop section
- Update contact information in the Contact section

#### Images
- Add your own images to the `assets/images/` folder
- Update image paths in the HTML
- Recommended sizes:
  - Product images: 400x300px
  - Hero illustrations: 400x400px
  - Service icons: 60x60px

#### Animations
- Modify animations in `script.js`
- Adjust timing, easing, and effects
- Add or remove animations as needed
